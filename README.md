■ 概要
「ClipSel_v2 Pro」は、圧倒的な保存容量と「キーボード完結」の操作性を追求した
クリップボード履歴・定型文管理ツールです。

最大10,000件の履歴を保持できる「実質無限」のキャパシティを備えながら、
独自の「ハーフサイズ・ダブルタブ」とIME対応の超高速リアルタイム検索により、
数千件の中から目的のデータを一瞬で引き出すことが可能です。

■ インストール・アンインストール
・インストール：適当なフォルダに解凍して実行してください。
・アンインストール：実行ファイルを削除してください。
※設定やデータは同フォルダの ClipSel_v2_Data.ini に保存されます。

■ 主な機能
1. 実質無限のクリップボード履歴（最大10,000件）
   日々の膨大なコピー内容を逃さず蓄積。過去の情報をいつでも再利用できます。
2. 大容量定型文（スニペット）管理（最大500件）
   頻繁に使う文章やコードを500件まで登録。素早く呼び出して貼り付けられます。
3. 究極のキーボード操作（IK/上下移動）
   ホームポジションを崩さず、[Ctrl + I] で上、[Ctrl + K] で下へリストを選択可能。
4. IME完全対応・リアルタイム検索
   日本語入力の確定前・確定後、またバックスペースでの削除にも完全追従。
   1文字ごとにリストが瞬時にフィルタリングされます。
5. ミニマル＆スマートUI
   ウィンドウサイズや位置を自動記憶。設定タブをあえて廃止し、
   [Tab]キー一つで「履歴」と「定型文」を最速で切り替えられる設計です。
6. スタートアップ管理
   タスクトレイ（右クリック）からWindows起動時の自動実行を簡単に設定。

■ 使用方法
1. ClipSel_v2_v2.6_Pro.exe を実行。タスクトレイに常駐します。
2. [Ctrl + Shift + U] でメイン画面を呼び出します（初回起動時に自動実行設定が可能）。
3. 検索窓で文字を入力し、[上下キー] または [Ctrl + I / K] で項目を選択。
4. [Tab] キーで「履歴」と「定型文」を切り替えられます。
5. [Escape] キーで画面を隠します。

■ ショートカットキー
・[Ctrl + Shift + U]：メイン画面の表示
・[Ctrl + I] / [Ctrl + K]：リストの上下移動
・[Tab] / [Shift + Tab]：タブ（履歴/定型文）の高速切り替え
・[Escape]：画面を隠す（常駐継続）
